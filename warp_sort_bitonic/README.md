## bitonic merge sort
this example implement warp level bitonic merge sort. The `med3` instruction, combined with `DPP`/`SHFL` makes it ultra fast and instruction saving to implement sorting on AMD GPU.

![image](res/bm-sort.png)

## build/run
```
sh rebuild.sh # build
./build/warp_sort.exe
```
